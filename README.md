# 🌐 Portfólio Web Profissional

Portfólio web responsivo desenvolvido com **HTML, CSS e JavaScript**, criado para apresentar informações profissionais, projetos, currículo para download e formas de contato. Ideal para **estágio ou vaga júnior em TI**.

---

## 📸 Preview
> Portfólio com layout moderno, animações, modo dark/light e otimizado para dispositivos móveis.

---

## 🚀 Funcionalidades

- 🎨 Design moderno e responsivo
- 🌙 Modo Dark / Light
- 📱 Compatível com celulares e tablets
- 🧑 Foto de perfil e plano de fundo
- 📂 Seção de projetos com cards animados
- 📄 Currículo em PDF para download
- ✉️ Formulário de contato
- 📧 Email clicável
- 📞 Telefone clicável
- 💬 Botão direto para WhatsApp
- 🌐 Links para redes sociais
- 🔍 SEO básico para indexação no Google
- ✨ Animações com ScrollReveal

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **ScrollReveal.js**
- **GitHub Pages** (publicação)
